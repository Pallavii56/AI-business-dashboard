# 📊 AI Business Dashboard

An interactive business analytics dashboard built using Streamlit and Python for analyzing CSV business data.

## 🚀 Features

* 📁 Upload CSV datasets
* 📌 KPI Dashboard
* 💰 Revenue Analytics
* 🏆 Top Categories Analysis
* 📈 Monthly Trend Visualization
* 📥 Export Processed Reports
* 🎨 Modern Dashboard UI

## 🛠 Tech Stack

* Python
* Streamlit
* Pandas
* Plotly

## 📂 Project Structure

AI_Business_Dashboard/

├── app.py
├── requirements.txt
├── sample_sales.csv
├── README.md
└── .gitignore

## ▶ Run Locally

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```

Open in browser:

```text
http://localhost:8501
```
